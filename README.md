# bash_Biomedical_Data_Retrieval
how shell scripting can be used to perform the tasks that This repository shows how to do a task that health and life science specialists may need to undertake to find and retrieve biomedical data and text. 

The objective of this repository is to show how to access biomedical databases provided by the [European Bioinformatics Institute (EBI)](https://www.ebi.ac.uk/) by using their [web services](https://www.ebi.ac.uk/services) for automated data and text retrieval.
